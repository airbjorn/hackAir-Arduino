Ethernet Library Reference

Library Reference

To use the Ethernet functions you must include hackair_ethernet.h after hackair.h and Ethernet.h.

#include <Ethernet.h>
#include <hackair.h>
#include <hackair_ethernet.h>

Constructor hackAirEthernet(EthernetClient ethernet, char *token)
Creates a container class for the Ethernet functions
Parameters:

  • ethernet: Ethernet shield’s library client
  • token: Pointer to the Authentication token used by the API

Usage:

// MAC Address for the ethernet shield
// Some shields should have the MAC address printed on them
byte mac[] = { 0xDE, 0xAD, 0xBE, 0xEF, 0xFE, 0xED };

// Initialize the Ethernet libraries
EthernetClient client;
hackAirEthernet hackAirNet(client, "AUTHORIZATION TOKEN"); // Write your API key here

Method int sendData(hackAirData &data)
Parameters:

  • &data: Data to send

Usage:

void loop() {
  // Check DHCP lease
  Ethernet.maintain();

  // Collect data
  struct hackAirData data;
  sensor.refresh(data);

  // Send with ethernet shield
  hackAirNet.sendData(data);
}
