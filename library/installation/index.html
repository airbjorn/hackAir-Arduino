Installation

Arduino IDE

The open-source Arduino IDE (Integrated Development Environment) is an easy-to-use software used for writing and uploading code to the Arduino boards. It is available for Windows, Mac OSX and Linux. Windows 10 users can optionally download it from the Microsoft store.

Download Arduino IDE (All platforms) Microsoft Store (Windows 10)

The Arduino website has a “Getting Started” guide on using the Arduino IDE here. We recommend getting familiar with the Arduino software before creating your first hackAir node.

hackAIR Library

The latest version of the hackAIR Arduino library is always available for download at the project’s Github page. The easiest way to add the library to your Arduino IDE is by using the latest release .ZIP file.

Library downloads Code repository Issues & Bugs

At the Arduino IDE, navigate to Sketch → Include Library → Add .ZIP library and select the file you just downloaded from the above link. The library and the example sketches should now be available.
